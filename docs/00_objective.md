> maximize our one truly scarce resource: human time and attention.

Agents are responsble for application logic, architectural design, maintenance, bug fixes, refactoring, code cleanliness and sustainability.

Humans are responsible for overall objective setting, scenario design, design review, and guidance on what needs to be built.

Agents and humans collaborate actively. Humans come with ideas and context. Agents help expand, critque, refine, and distil those ideas. Convergence occurs iteratively.

Code does not get written until a detailed plan is in place. A detailed planning process includes: ideation, reseach of the ideas, collaborative design and planning, agentic design review, detailed spec generation based on design output, agentic spec review, breakdown of specs into tasks, agentic review of tasks, and consolidation of all previous steps into formal documentation. That documentation is then passed to an implementation engine.

Implementation includes: application and library code generation, test generation (any applicable: unit, property, integration, E2E, contract, scenario, digital twin), compilation and test execution, code and architectural review, and addressing **all agreed upon** review issues.

Telemetry, metrics, observability are first class principles to be deeply integrated.
