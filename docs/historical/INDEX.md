# docs/historical/ — Quarantined-but-Preserved Audit-Trail Artifacts

This directory holds documentation that is **clearly historical** — superseded plans, inception-era roadmaps, one-time validation execution logs, and other audit-trail artifacts — separated out of the live `docs/` tree **without deletion**. Nothing here is normative or current; it is kept for provenance and to preserve the project's audit trail. For current state always consult [STATUS.md](../../STATUS.md), [docs/INITIATIVES.md](../INITIATIVES.md), and the live specs under [specs/](../../specs/).
